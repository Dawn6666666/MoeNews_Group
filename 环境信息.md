## 2. 当前项目结构

目前项目已经完成基础开发，具体结构如下：

* sql：存放数据库初始化脚本。
  * 脚本文件：d:\Code\IDEA\MoeNews\sql\init.sql。
  * 内容：创建 `moenews` 数据库、用户表、新闻分类表、新闻表、评论表，并写入默认用户、分类和示例新闻。
* backend：后端 Spring Boot 3.x 项目工程。
  * 项目配置文件：d:\Code\IDEA\MoeNews\backend\pom.xml (已引入 Web、Validation、MyBatis-Plus、MySQL、Lombok、JWT 和加密等依赖)。
  * 主启动类：d:\Code\IDEA\MoeNews\backend\src\main\java\com\moenews\MoeNewsApplication.java。
  * 配置文件：d:\Code\IDEA\MoeNews\backend\src\main\resources\application.yml (已配置数据库连接、MyBatis-Plus 选项和本地上传路径)。
  * 主要包结构：
    * `common`：统一返回结果。
    * `config`：MyBatis-Plus、Web MVC、跨域、上传静态资源映射配置。
    * `controller`：认证、前台新闻、后台用户/分类/新闻/评论/上传/统计接口。
    * `dto`：请求参数对象。
    * `entity`：数据库实体。
    * `exception`：业务异常和全局异常处理。
    * `mapper`：MyBatis-Plus Mapper。
    * `security`：JWT 工具、登录用户上下文、自定义拦截器。
    * `service`：业务逻辑。
    * `vo`：接口响应视图对象。
* frontend：前端 Vue 3 + Vite 项目工程。
  * 已完成前台新闻展示、登录注册、个人资料、后台统计、用户管理、分类管理、新闻管理、评论管理、Markdown 编辑和图片上传页面。
  * 主要目录：
    * `src/api`：Axios 请求封装。
    * `src/layout`：前台和后台布局。
    * `src/router`：Vue Router 路由与访问控制。
    * `src/stores`：Pinia 登录状态。
    * `src/utils`：状态选项、资源 URL 工具。
    * `src/views`：前台和后台页面。
* uploads：本地上传文件目录。
  * 该目录由运行时生成，已在 `.gitignore` 中忽略，不提交到 Git。

## 3. 集成开发环境导入与开发指南

### 3.1 导入项目步骤

1. 打开 IntelliJ IDEA，选择打开项目 (Open)。
2. 选择 MoeNews 的根目录 `D:\Code\IDEA\MoeNews` 确认打开。
3. 打开后，IDEA 会自动检测并提示加载 Maven 模块，点击确认或等待 Maven 自动同步依赖即可。

### 3.2 数据库连接与建表

1. 本机 MySQL 服务（MySQL80）已在 3306 端口运行，请使用您的客户端工具（如 Navicat，或 IntelliJ IDEA 右侧的 Database 面板）连接到该服务。
2. 连接账号为 `root`，密码为 `114514`。
3. 连接成功后，打开并运行位于 `d:\Code\IDEA\MoeNews\sql\init.sql` 的脚本，它将自动创建名为 `moenews` 的数据库以及相关的用户、分类、新闻和评论数据表，并导入初始的管理员、普通用户和新闻分类数据。

也可以在 PowerShell 中执行：

```powershell
& 'C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe' --default-character-set=utf8mb4 -uroot -p114514 -e "source D:/Code/IDEA/MoeNews/sql/init.sql"
```

注意：不要使用 PowerShell 的 `< sql\init.sql` 重定向方式导入，因为 PowerShell 不支持该 Bash 写法，且容易产生中文字符集问题。

### 3.3 修改后端配置文件

当前 [application.yml](file:///d:/Code/IDEA/MoeNews/backend/src/main/resources/application.yml) 已配置：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/moenews?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: 114514
```

如果迁移到其他机器，需要同步修改 `spring.datasource.username`、`spring.datasource.password` 和 `moenews.upload.path`。

### 3.4 后端启动与运行

1. 在 IntelliJ IDEA 中定位到 [MoeNewsApplication.java](file:///d:/Code/IDEA/MoeNews/backend/src/main/java/com/moenews/MoeNewsApplication.java) 文件。
2. 右键点击并选择运行，后端服务将启动并监听 8080 端口。

也可以使用 IDEA 内置 Maven 在 PowerShell 中启动：

```powershell
cd D:\Code\IDEA\MoeNews\backend
& 'D:\Jetbrains\IntelliJ IDEA 2025.2.4\plugins\maven\lib\maven3\bin\mvn.cmd' spring-boot:run
```

后端启动后可访问：

* 前台新闻接口：http://localhost:8080/api/news
* 公开分类接口：http://localhost:8080/api/categories
* 后台统计接口：http://localhost:8080/api/admin/dashboard/statistics

### 3.5 前端依赖安装与启动

1. 在集成开发环境的内置终端窗口中，将工作目录切换到 `frontend`。
2. 执行以下命令安装前端依赖：
   ```bash
   npm.cmd install
   ```
3. 安装完毕后，执行以下命令启动前端开发服务器：
   ```bash
   npm.cmd run dev
   ```
4. 启动完成后，可通过浏览器访问前端开发控制台（通常为 http://localhost:5173）。

当前建议访问地址：

* 前端：http://127.0.0.1:5173
* 后端：http://localhost:8080


### 5.2 前端构建检查

```powershell
cd D:\Code\IDEA\MoeNews\frontend
npm.cmd run build
```

### 5.3 后端接口冒烟测试

管理员登录：

```powershell
Invoke-RestMethod -Uri http://localhost:8080/api/auth/login `
  -Method Post `
  -ContentType 'application/json; charset=utf-8' `
  -Body '{"username":"admin","password":"123456"}'
```

前台新闻列表：

```powershell
Invoke-RestMethod -Uri http://localhost:8080/api/news -Method Get
```

### 5.4 检查端口占用

```powershell
Get-NetTCPConnection -LocalPort 8080,5173 -State Listen -ErrorAction SilentlyContinue |
  Select-Object LocalPort,OwningProcess
```

### 5.5 停止本地服务

```powershell
Get-NetTCPConnection -LocalPort 8080,5173 -State Listen -ErrorAction SilentlyContinue |
  ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }
```
